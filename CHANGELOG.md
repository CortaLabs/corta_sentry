# Changelog

## 0.1.0 - 2026-07-12

- Initial modular-monolith implementation.
- Fail-closed target/port scope engine with mapped-IPv4 normalization and audit decisions.
- SQLite WAL migrations, immutable observations/audit, assets, candidates, findings, changes, jobs, sessions, and rule loads.
- Bounded TCP, banner, HTTP, TLS, neighbor, mDNS, and SSDP collection foundations.
- Conservative strong-identifier asset resolver and deterministic typed YAML fingerprint rules.
- Bounded Nmap XML, Zeek JSONL, Suricata EVE, and Nuclei JSONL importers.
- Authenticated REST API, CSRF-protected browser sessions, metrics, embedded React console, fixture demo, installer, Docker, and CI.
- Official MCP toolset for Codex and Claude, versioned observation JSONL export, and Linux passive host-connection snapshots.
- Fixed empty-job JSON responses that broke the Scans view, custom-config path resolution, terminal queued cancellation, and strong-identity continuity conflicts.
